# Privacy Policy for [! Moderator#5604]

**Last Updated:** February 27, 2026

## Introduction

This Privacy Policy explains how [Имя бота] ("the Bot") collects, uses, and protects information when you use our services on Discord.

## Information We Collect

### Automatically Collected Data

The Bot automatically collects and stores the following information:

- **User IDs**: Discord user identification numbers
- **Server IDs**: Discord server (guild) identification numbers
- **Role Information**: User roles and role changes
- **Message Metadata**: Message IDs, channel IDs, timestamps (for moderation logging)
- **Voice Activity**: Voice channel join/leave events
- **Account Information**: Account creation dates, usernames, nicknames

### Data Stored in Database

The Bot stores the following data in its database:

- **Moderation Records**: Warnings, bans, mutes, and other moderation actions
- **Staff Information**: Staff member statistics and activity
- **Verification Data**: User verification status and assigned roles
- **Appeal Records**: Ban and restriction appeal information
- **Backup Data**: Automated database backups

## How We Use Your Information

We use collected information for:

- **Moderation**: Logging moderation actions and maintaining server security
- **Verification**: Managing user verification and role assignment
- **Statistics**: Tracking server activity and staff performance
- **Appeals**: Processing user appeals for bans and restrictions
- **Logging**: Maintaining audit logs for server administrators
- **Automation**: Automated role management and account age verification

## Data Storage and Security

- All data is stored on a secure MySQL database
- Database is hosted on a private server (192.168.0.11)
- Access to the database is restricted to authorized administrators only
- Automated backups are created daily
- Data is retained as long as necessary for moderation purposes

## Data Sharing

We do NOT:
- Sell your data to third parties
- Share your data with external services
- Use your data for advertising purposes

We MAY share data:
- With server administrators and moderators for moderation purposes
- When required by Discord Terms of Service or applicable law

## Your Rights

You have the right to:

- **Access**: Request a copy of your stored data
- **Deletion**: Request deletion of your data (subject to moderation record retention)
- **Correction**: Request correction of inaccurate data

To exercise these rights, contact us through our support server.

## Data Retention

- **Moderation Records**: Retained indefinitely for security purposes
- **Verification Data**: Retained while you are a member of the server
- **Logs**: Retained for up to 90 days
- **Backups**: Retained for up to 30 days

## Children's Privacy

The Bot does not knowingly collect information from users under 13 years of age. If you are under 13, please do not use the Bot.
